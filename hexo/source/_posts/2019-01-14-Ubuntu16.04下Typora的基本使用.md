#### 1. 缩进
```
&emsp;	# 全角空格
&ensp;	# 半角空格
```



#### 2. 数学公式(Latex)

##### （1）上下标

使用‘_'下标，使用'^‘上标，上下标含有多个，利用大括号包住；

##### （2）公式中的空格

```
\quad	# 1m的宽度
\qquad	# 2m的宽度
a \ b	# 1/3m的宽度，a、b之间
```

##### (3) 分数

```
\dfrac{1}{2}
```

##### （4）求和

```
\sum_{i=1}^{N}
```

##### （5）范数

```
\left\| x-y \right\|
```





#### 3. 换行

```
<br>	# 在每行后面加上给标签
```
